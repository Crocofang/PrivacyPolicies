# Политика конфиденциальности

## 1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки информации при использовании приложения городского исторического квеста (далее — «Приложение»).

Используя Приложение, пользователь подтверждает согласие с настоящей Политикой.

## 2. Какие данные используются

Приложение не требует создания учётной записи и не запрашивает у пользователей имя, адрес электронной почты, номер телефона или иные персональные данные.

Для работы отдельных функций Приложение может получать данные о местоположении устройства пользователя.

## 3. Для чего используется местоположение

Данные о местоположении используются исключительно для:

* определения нахождения пользователя рядом с игровыми точками маршрута;
* проверки выполнения заданий, связанных с посещением определённых мест;
* обеспечения корректной работы механики квеста.

## 4. Хранение и передача данных

Если иное не указано в Приложении, данные о местоположении используются только во время работы квеста и не передаются третьим лицам.

Организаторы не продают и не предоставляют данные пользователей рекламным или маркетинговым организациям.

## 5. Разрешения устройства

Для корректной работы отдельных функций Приложение может запрашивать разрешение на доступ к геолокации устройства.

Пользователь вправе в любой момент отключить доступ к геолокации в настройках устройства. При этом отдельные функции Приложения могут работать некорректно или стать недоступными.

## 6. Безопасность

Организаторы принимают разумные меры для обеспечения безопасности информации, используемой в работе Приложения.

## 7. Изменения политики

Организаторы вправе вносить изменения в настоящую Политику конфиденциальности. Актуальная версия Политики публикуется в Приложении.

## 8. Контакты

По вопросам, связанным с работой Приложения и настоящей Политикой конфиденциальности, пользователь может обратиться к организаторам через контактные данные, указанные в Приложении.
